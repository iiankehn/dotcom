Welcome to the Coldstone Public Beta. Coldstone v2 is the next version of the Coldstone CMS for iiankehn.com. This release includes a maltitude of new features and user experience improvements.

Below is the official changelog for this release.

New User Experience

This release includes an array of user experience changes based on user feedback from the 1.0 release in 2023. This release includes a streamlined user interface designed to get you where you need to be without having to navigate through more pages.

It also includes a universal navigation optimized for both desktop and mobile, and includes mobile-specific theming if you use Safari or Chrome.
