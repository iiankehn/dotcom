Sevan vNext Release 20251018

Posted on Oct 18, 2025 by Iian Kehn

The next version of Sevan vNext is live. This is the first initial release of the new codebase, so features may be missing, inaccurate, not working, or broken in this release.

What's new

This release features a new design system, focusing on new visuals, and emphasizing content. This release is available for both desktop and mobe out of the box and does not require additional setup.

What's coming

We are working on a system to feature video content without breaking the existing layout. We will be opening up a preview of this feature in a future release and are looking to generate feedback when the time comes.
